# DiabRetina AI Frontend

DiabRetina AI：糖尿病性视网膜病变诊断智能平台（前端应用程序）。

> ***Relevant course***
> * Comprehensive Project in Specialized Direction 2025 (2025年同济大学专业方向综合项目)

## 项目组成

* `/public`
项目静态资源

* `/src`
项目源代码

* `jsconfig.json`
JavaScript 项目配置文件

* `package-lock.json`
项目配置文件（锁定依赖版本）

* `package.json`
项目配置文件

## 启动程序

```bash
npm install
npm run start
```
